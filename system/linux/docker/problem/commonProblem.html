网络被占用问题解决

问题复现：docker run时，报网络被占用问题

docker: Error response from daemon: endpoint with name *** already exists in network bridge.

查看当前网络下，有哪些服务挂载

docker network inspect bridge

找到冲突的服务网络，卸载掉

docker network disconnect --force bridge ***
