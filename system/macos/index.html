# 三、macOS 时光机器备份加速命令

  • 如果你真的打算让时光机器全力全速工作，那也是有办法的，今天就教大家通过命令行，用命令强制关闭系统对时光机器的限流，俗称“解除封印”。打开终端，输入以下命令：
sudo sysctl debug.lowpri_throttle_enabled=0
1
  • 这时你就会发现时光机器的备份速度变快很多很多了！！基本能达到网络和硬盘读写的应有的速度了。等它完成了首次的备份之后，你可以再执行下面的命令，恢复到原本限流的状态，以保证日后使用电脑时不被时光机器备份占去太多的资源导致变卡。
sudo sysctl debug.lowpri_throttle_enabled=1
1

# 2、放入空白的光盘（也可以待会儿再放，just you like _）
# 3、按command+空格键，输入“ter" ，以打开Mac的终端（terminal）
