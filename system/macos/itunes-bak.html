# 🍎 一、默认备份位置

在 macOS 中，iPhone 的备份路径如下：

~/Library/Application\ Support/MobileSync/Backup
1

也就是：

/Users/你的用户名/Library/Application Support/MobileSync/Backup/
1

💡“~” 表示当前用户主目录。

你可以在 Finder 中按下 Command + Shift + G，然后输入路径跳转。

# ⚙️ 二、更改备份位置的方法（推荐）

# 1️⃣ 关闭 iTunes（或 Finder 同步窗口）

确保没有正在进行的备份或同步。

# 2️⃣ 重命名原备份文件夹

打开终端，执行：

mv ~/Library/Application\ Support/MobileSync/Backup ~/Library/Application\ Support/MobileSync/Backup_old
1

（这只是做个备份，以防出错。）

# 3️⃣ 创建新的备份目标路径

例如，你想把备份放在外置硬盘 /Volumes/MyDisk/iPhoneBackup：

mkdir -p /Volumes/MyDisk/iPhoneBackup
1

# 4️⃣ 创建符号链接

然后在终端执行：

ln -s /Volumes/MyDisk/iPhoneBackup ~/Library/Application\ Support/MobileSync/Backup
1

这一步是关键，它让系统以为备份还在原路径，但实际上数据会写入新的路径。

# ✅ 三、验证是否成功

执行以下命令检查链接：

ls -l ~/Library/Application\ Support/MobileSync/
1

应能看到类似：

Backup -> /Volumes/MyDisk/iPhoneBackup
1

表示链接生效。

# 📱 四、测试

打开 iTunes（或 Finder）连接 iPhone，执行一次备份。 完成后去 /Volumes/MyDisk/iPhoneBackup 看看是否生成新的备份文件夹。

# ⚠️ 注意事项

  • 确保外接磁盘在备份时已挂载（否则备份会失败）。

  • 如果更换磁盘路径，旧备份不会自动移动，可以手动复制过去。

  • 建议使用 HFS+ 或 APFS 格式磁盘，不要用 FAT/exFAT（权限不兼容可能出错）。
