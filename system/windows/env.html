      # 💻 Windows 常见环境变量速查表

      变量名 含义说明 示例路径
      %userprofile% 当前用户主目录 C:\Users\Alice
      %username% 当前登录用户名 Alice
      %appdata% 当前用户的 Roaming 配置目录 C:\Users\Alice\AppData\Roaming
      %localappdata% 当前用户的 Local 配置目录 C:\Users\Alice\AppData\Local
      %programdata% 所有用户的公共程序数据目录 C:\ProgramData
      %temp% / %tmp% 临时文件目录 C:\Users\Alice\AppData\Local\Temp
      %homepath% 用户主目录路径（不含盘符） \Users\Alice
      %homedrive% 用户主目录盘符 C:
      %systemroot% Windows 系统安装目录 C:\Windows
      %windir% 同 %systemroot%，系统目录 C:\Windows
      %path% 系统可执行程序搜索路径 多个路径，用 ; 分隔
      %computername% 当前计算机名 MY-PC
      %userdomain% 当前用户所属的域或计算机名 MY-PC
      %public% 公共用户目录 C:\Users\Public
      %commonprogramfiles% 公共程序文件目录 C:\Program Files\Common Files
      %programfiles% 程序默认安装目录 C:\Program Files
      %programfiles(x86)% 32位程序默认安装目录 C:\Program Files (x86)
      %allusersprofile% 所有用户共享配置目录 C:\ProgramData
      %comspec% 命令解释器路径 C:\Windows\System32\cmd.exe
      %number_of_processors% CPU 核心数 8
      %processor_architecture% CPU 架构 AMD64
      %os% 操作系统类型 Windows_NT

      📘 提示：

      • 在 CMD 或 PowerShell 中输入：
        echo %变量名%
      1

      即可查看具体值，例如：

      echo %userprofile%
      1
      • 在文件资源管理器地址栏中也可直接输入 %变量名% 跳转。
