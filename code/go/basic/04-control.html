04、流程控制

if / for / switch

for i := 0; i < 3; i++ {
    fmt.Println(i)
}

switch code {
case 200:
    fmt.Println("ok")
default:
    fmt.Println("fail")
}

for-range 注意点

for i, v := range []string{"a", "b"} {
    fmt.Println(i, v)
}

常见坑

  1. 误以为 Go 有 while，实际上只有 for。
  2. 在循环里启动协程时直接引用循环变量。

建议

  1. 复杂分支用 switch 提升可读性。
  2. 提前返回（guard clause）减少嵌套。
