Java

Java 是一门面向对象、跨平台、生态成熟的编程语言，适合后端服务、企业应用、中间件和 Android 等场景。

一、先准备好运行环境

1、检查本机 Java 环境

java -version
javac -version

2、如果未安装（Ubuntu / Debian）

sudo apt update
sudo apt install -y openjdk-17-jdk
java -version
javac -version

3、如果未安装（CentOS / Rocky / AlmaLinux）

sudo dnf install -y java-17-openjdk java-17-openjdk-devel
java -version
javac -version

二、3 分钟跑通第一个 Java 程序

1、新建文件 Hello.java

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}

2、编译并运行

javac Hello.java
java Hello

3、预期输出

Hello Java

三、日常最常用命令（可直接复制）

1、查看 JVM 参数

java -XX:+PrintFlagsFinal -version | head -n 30

2、打包成可执行 jar

# 先编译
javac -d out src/main/java/com/demo/Main.java

# 生成 jar（入口类改成你自己的）
jar --create --file app.jar --main-class com.demo.Main -C out .

3、运行 jar 并指定环境参数

java -Xms256m -Xmx512m -Dspring.profiles.active=prod -jar app.jar

四、学习路线建议

  1. Java 基础语法与 OOP（变量、流程、类、接口）
  2. 集合与泛型（List/Map/Set、泛型边界）
  3. 并发（线程池、锁、并发容器）
  4. JVM 基础（内存模型、GC、调优）
  5. Spring 生态（Spring Boot、Spring Cloud）
